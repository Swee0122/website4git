Git Commands you need to know

Git cheatsheet
  • git init
    Initializes a new Git repository in your current directory.
  • git status
    Displays the state of the working directory and staging area.
  • git add -A
    Stages all changes made since the last commit.
  • git commit -m ""
    Commits all staged changes with a customized message.
  • git log
    Displays the commit history of a repository.
  • git remote add origin (GitHub repo URL)
    Connects your local repository to one on GitHub.
  • git push
    Pushes all of your commits/changes to the connected GitHub repository.
  • git pull
    Pulls the remote repository and any changes that may have been made to it.
  • git clone
    Creates a copy of the linked repository on your local device.
  • git branch
    Creates a branch within your repository which is the same but separate allowing you to experiment with and test new features without breaking your code.
