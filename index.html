Git Commands you need to know

  • git init
    Initializes a new Git repository in your current directory.
  • git status
    Displays the state of the working directory and staging area.
  • git add -A
    Stages all changes made since the last commit
  • git commit -m ""
    Displays the state of the working directory and staging area.
  • git log
    Displays the state of the working directory and staging area.
